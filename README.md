# Elphie

Elphie is a framework for making slides.
It is based on Python and Inkscape.

Example: [example.pdf](example/example.pdf), [example.py](example/example.py)

# Requirements

* Python 3.2 or newer
* Inkscape
* pdftk
* pygments
* lxml

# Installation

    python setup.py build
    python setup.py install
